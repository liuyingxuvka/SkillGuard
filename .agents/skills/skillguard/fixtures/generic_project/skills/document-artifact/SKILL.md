# Document Artifact Fixture

## Purpose

Exercise a target-owned document structure and render workflow without transferring document judgment or rendering authority to SkillGuard.

## Use When

Use only in SkillGuard declared-check and project-adoption fixtures.

## Do Not Use When

Do not use for real document production or release claims.

## Required Workflow

1. Build the declared document structure through the native fixture route.
2. Render and inspect the artifact through the native fixture check.
3. Preserve separate structure and render evidence.

## Hard Gates

- A heading-only file is shallow and must not pass.
- Structure evidence cannot substitute for render evidence.
- SkillGuard may consume receipts but may not render or judge the document itself.

## Output Requirements

Report structure, render, gaps, and claim boundary separately.
