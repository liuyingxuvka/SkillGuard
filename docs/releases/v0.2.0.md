# SkillGuard v0.2.0

SkillGuard v0.2.0 turns the project from a primarily declarative maintenance checker into a FlowGuard-backed executable-contract runtime while preserving the existing V1 surface as a migration and compatibility layer.

## Highlights

- Compile a current FlowGuard behavior model plus confirmed target bindings into an exact executable contract and check manifest.
- Claim target-local runs with typed routes, write locks, hash-chained events, immutable evidence receipts, replay, bounded loops, conditional skips, artifact validation, and monotonic closure profiles.
- Treat AI completion as evidence submission; hard checks, witnessed tool actions, or versioned judgment rubrics own pass authority.
- Fingerprint the SkillGuard runtime so a Guard behavior change creates a new run and makes affected earlier evidence stale.
- Recover locks from exited or verified-failed writers with an audit event while continuing to block a live overlapping writer; idempotent resume reacquires its locks.
- Stage and verify the whole installed skill tree, activate atomically, retain the previous copy, and roll back automatically if post-activation validation fails.
- Audit canonical/installed/Git/release provenance and scan tracked plus unignored public candidates for secrets, machine paths, runtime state, blocked private types, and unreviewed images.

## Upgrade

Use the staged installer from the repository root. The stage path must end in `.codex/skills/skillguard`.

```powershell
python .agents/skills/skillguard/scripts/skillguard_install.py --canonical-skill-root .agents/skills/skillguard --stage-root <staged-codex-home>/.codex/skills/skillguard --codex-home <codex-home> --prepare
python .agents/skills/skillguard/scripts/skillguard_install.py --canonical-skill-root .agents/skills/skillguard --stage-root <staged-codex-home>/.codex/skills/skillguard --codex-home <codex-home> --activate
```

Activation preserves the previous active copy under the Codex backup root. A post-activation smoke or parity failure restores that copy automatically.
